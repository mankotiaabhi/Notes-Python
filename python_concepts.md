# Core Python Concepts

## 1. Variables

```python
x = 10
name = "Abhay"
```

---

## 2. Conditional Statements

```python
if x > 5:
    print("Greater")
else:
    print("Smaller")
```

---

## 3. Loops

```python
for i in range(5):
    print(i)

while x > 0:
    x -= 1
```

---

## 4. Functions

```python
def add(a, b):
    return a + b

print(add(2, 3))
```

---

## 5. OOP Concept

```python
class Person:
    def __init__(self, name):
        self.name = name

    def greet(self):
        print("Hello", self.name)

p = Person("Abhay")
p.greet()
```
